- **布局**: "深色模式" 终端窗口.
- **窗口 1: 安装 (一键脚本)**
  ```bash
  # 方式 A: 一键脚本
  curl -fsSL https://clawd.bot/install.sh | bash
  ```
- **窗口 2: 安装 (NPM)**
  ```bash
  # 方式 B: NPM
  npm i -g clawdbot@latest
  ```
- **窗口 3: 初始配置流程**
  ```bash

  clawdbot onboard --install-daemon
  # 2. 浏览器访问控制台
  http://127.0.0.1:18789/
  ```
- **提示框**: "排障小贴士: 没反应？检查 Pairing 批准。远程？用 SSH Tunnel，别裸奔！"

### 给设计师的实现说明

1.  **代码语法高亮**: 确保终端命令 (Canvas 2 第 4 区) 对命令 (`curl`, `npm`) 和参数使用不同颜色，使其看起来像真实的终端.
2.  **图标**: 使用轮廓风格图标，线条略微 "不完美" 以匹配手绘美学.
3.  **空间管理**: Canvas 2 第 4 区 (代码) 需要垂直空间. 如果文本太长，截断 URL 视觉显示但保持命令清晰.
4.  **一致性**: 在 Canvas 2 中使用与 Canvas 1 页头相同的 "机器人" 角色来引导用户 (例如，在 "缺点" 中拿着警告标志).
