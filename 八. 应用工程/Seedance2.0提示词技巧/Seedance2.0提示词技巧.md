# Seedance 2.0 提示词技巧（可直接复制）

这是一套 Seedance 2.0 常用的提示词写法与模板，适配文生视频、图生视频、多镜头分镜、运镜与角色稳定。目标是：**用更少、更明确的文字，让模型更可控**。

## 0. 快速上手（先把“可控三件套”写清楚）

- **主体是谁**：人物/物体的关键特征（年龄气质、服装配饰、材质形态）。
- **动作怎么动**：动作节奏与连续性（慢、连贯、自然、少转头少高速）。
- **镜头怎么拍**：景别 + 运镜方式 + 稳定性（固定/平移/跟拍/推近，强调无抖动）。

剩下的（风格、画质、氛围）放后面补，不要喧宾夺主。

## 1. 万能结构（最小可控单元）

主体 + 动作 + 场景 + 光影 + 镜头/运镜 + 风格 + 画质 + 约束

示例：

> 一位年轻女生在海边缓慢行走，微风轻拂头发，自然微笑看向镜头，黄昏暖光，中景，缓慢推近，稳定跟拍，无抖动无闪烁，画面流畅，细节清晰，电影感。

## 2. 动作描述：写“连续动作链”，别写一个词

### 2.1 好用写法

- 用 **连续动作链**：A → B → C（每步都慢一点、自然一点）。
- 用 **身体部位 + 小动作**：眼神、手指、肩颈、步伐，效果往往更稳。
- 用 **节奏词**：缓慢、轻柔、连贯、自然、流畅、不僵硬、微微、轻轻。

动词组合（可直接复制）：

- 缓慢转身、轻轻抬手、脚步轻移、微微低头、随风摆动
- 停顿一秒后转头看向镜头、眨眼、轻笑、抬手整理头发

### 2.2 容易翻车的写法（尽量少用）

- 高速/剧烈：狂奔、激烈打斗、连续空翻、疯狂甩头
- 人太多/互动太复杂：多人打架、拥挤人群混战、复杂肢体纠缠
- 大幅度形变：夸张拉伸、身体扭曲、脸部快速表情变化

## 3. 镜头与运镜：用“摄影机语言”写，稳定性要写死

### 3.1 景别（任选其一）

- 特写 / 近景 / 中景 / 全景 / 远景

### 3.2 运镜（从简单开始）

- 缓慢推近 / 轻微拉远 / 平稳平移 / 稳定跟拍 / 轻微环绕半圈
- 固定镜头 / 手持但稳定 / 云台稳定 / 无抖动

示例：

> 中景，缓慢推近，稳定跟拍，画面丝滑不卡顿，无抖动无果冻。

## 4. 角色稳定：先写“角色卡”，再写“稳定约束”

### 4.1 角色卡（建议每次都复用）

把人物的“不可变特征”写成一段固定文本，后面每个镜头都带上（尤其是多镜头）。

示例（可复制修改）：

> 同一位年轻女生，黑色齐肩短发，浅色针织衫，银色耳环，五官清晰，面部稳定不变形，服装与发型保持一致。

### 4.2 稳定约束（必加）

- 五官清晰、面部稳定、不扭曲、不变形
- 人体结构正常、比例自然、手部正常、动作不僵硬
- 同一角色、服装一致、发型不变、配饰不变

小技巧：

- 如果出现“变脸”，通常先 **降低动作强度**，再 **提高约束密度**。
- 多镜头时，把“同一角色、服装一致、面部稳定”写成全片总约束，并在每个镜头里重复一次关键短句。

## 5. 画质与光影：用“可观察”的词

画质词（复制即可）：

- 高清、细节丰富、锐度清晰
- 电影质感、色彩自然、光影柔和
- 无模糊、无重影、无闪烁、画面稳定

光影氛围（任选其一）：

- 黄昏暖光 / 阴天柔光 / 室内窗边侧光 / 霓虹夜景反射光

## 6. 风格与氛围：用“风格 + 色调 + 情绪”

- 治愈清新、唯美文艺、日系清新、韩系氛围感
- 赛博朋克、暗调高级、复古胶片、梦幻柔光
- 极简干净、高级质感、真实实景

## 7. 多镜头/分镜：按时间写“镜头块”，每块都闭环

推荐写法：

- 每个镜头 2–4 秒（更容易稳）。
- 每个镜头都写：**景别 + 运镜 + 动作链 + 稳定约束**。
- 结尾加“衔接要求”：丝滑转场、节奏一致、色调统一。

模板（可复制）：

```text
全片约束：同一角色，服装发型一致，面部稳定不变形，人体比例正常；画面稳定无抖动无闪烁；整体色调统一。

镜头1（0-3s）：近景，缓慢推近。人物抬眼看向镜头，轻轻微笑，抬手整理头发，动作自然连贯。
镜头2（3-6s）：中景，平稳跟拍。人物缓慢行走，脚步轻移，微风拂动头发，表情保持一致。
镜头3（6-9s）：全景，轻微拉远。人物停下回头，短暂停顿，镜头定格一秒后结束。
```

## 8. 参考素材写法：把“参考关系”写清楚（如果你的工作流支持 @素材）

很多人会把“控制难题”拆开：用参考图锁定角色/服装，用参考视频锁定动作与运镜，用文字补齐剧情与约束。关键是：**在提示词里明确每个素材负责什么**。

模板（可复制）：

```text
参考 @图片1 的人物长相与服装（全程保持一致），参考 @视频1 的运镜节奏（缓慢推近 + 轻微环绕），场景参考 @图片2 的海边黄昏氛围。
人物动作：缓慢行走 → 停下 → 转头看镜头 → 微笑。
约束：面部稳定不变形，人体结构正常，动作自然不僵硬；画面稳定无抖动无闪烁，无重影。
```

如果素材很多，提示词反而要更短：只写“参考关系 + 动作链 + 总约束”，别堆风格词。

## 9. 避雷清单：这些写法容易把模型带偏

- 模糊词：好看、很美、很酷（信息量太低）
- 自相矛盾：超高速 + 极度稳定、疯狂甩头 + 面部绝对稳定
- 角色/互动过载：多人复杂互动、打架、拥挤群像（更难稳定）
- 高风险内容：色情、暴力、危险动作、版权人物（合规风险）

## 10. 排错指南：哪里不对改哪里

| 现象             | 常见原因                   | 提示词怎么改                                                                      |
| ---------------- | -------------------------- | --------------------------------------------------------------------------------- |
| 画面抖动/果冻感  | 运镜太复杂或节奏太快       | 改成“固定镜头/平稳平移/稳定跟拍”，加“无抖动无果冻”，降低动作幅度                  |
| 变脸/长相漂移    | 角色描述太少或镜头切换太猛 | 增加“角色卡”，每镜头重复“同一角色、服装一致、面部稳定不变形”，减少快速转头/大表情 |
| 手部/肢体畸变    | 动作太复杂或多人交互       | 简化为“小动作链”，加“手部正常、人体结构正常”，减少遮挡与交互                      |
| 镜头不按你写的走 | 运镜描述太多太杂           | 只留一个运镜（推近/平移/跟拍其一），并写“保持全程一致”                            |
| 画面糊/重影/闪烁 | 细节/画质约束不足          | 增加“细节清晰、无模糊无重影无闪烁、画面稳定”，减少高频运动                        |

## 11. 直接可复制模板

### 模板 1：唯美人像短视频

> 同一位年轻女生，黑色齐肩短发，浅色针织衫，银色耳环，面部稳定不变形，服装发型一致。林间小路，暖光柔和。中景，缓慢推近，稳定跟拍。人物缓慢行走，抬手整理头发，停下看向镜头，自然微笑。画面流畅稳定，无抖动无闪烁，无重影，细节清晰，电影感。

### 模板 2：氛围感风景

> 海边日落，海浪轻拍沙滩，天空暖橙色调，光影柔和。全景，镜头缓慢平移，画面丝滑稳定，无抖动无闪烁，无重影。细节清晰，高级质感。

### 模板 3：图生视频（首帧参考）

> 以参考图为准保持人物样貌与服装一致。动作：缓慢抬手 → 停顿 → 转身看向镜头，动作自然连贯不僵硬。中景，稳定跟拍，轻微推近。约束：面部稳定不变形，人体结构正常，画面稳定无抖动无闪烁，高清细节，电影质感。

### 模板 4：产品展示（电商短视频）

> 桌面产品特写，材质细节清晰，反射自然。特写 → 中景，镜头缓慢推近后轻微拉远，平稳无抖动。产品缓慢旋转展示，停顿一秒突出 LOGO/纹理。画面干净极简，色彩准确，无闪烁无重影，细节锐利清晰。

## 12. 十条速记

1. 主体写清楚：角色卡先写好，反复复用。
2. 动作写慢：用连续动作链，不用一个词糊弄。
3. 运镜写稳：从固定/平移/跟拍开始。
4. 约束写死：不变形、不抖动、不闪烁、无重影。
5. 多镜头写块：时间段 + 景别 + 运镜 + 动作链 + 约束。
6. 少写矛盾：想要“稳”，就别写“狂飙”。
7. 少堆风格词：先可控，再好看。
8. 一次只改一处：方便定位是哪里导致的翻车。
9. 复杂动作靠参考：能用参考视频就别纯靠文字硬控。
10. 先短再长：短片段跑通后再拼接成成片。
