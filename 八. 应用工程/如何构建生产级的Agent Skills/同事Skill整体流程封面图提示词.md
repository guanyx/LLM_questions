# 同事 Skill 整体流程封面图提示词

Create an eye-catching cover image for social media in hand-drawn technical sketch style on pure white background. The theme is "How a Colleague Is Distilled into an AI Skill".

Use a high-level workflow metaphor rather than a detailed technical diagram. The image should quickly show the full pipeline from collecting a colleague's information to generating a runnable Skill. Keep the content minimal, clean, and easy to grasp at a glance.

**Design concept:**
Use a left-to-right transformation flow:

- Left: colleague information and source materials
- Center: two parallel distillation branches
- Right: merged final Skill asset

The overall feeling should be "把一个真实同事，压缩成一个可调用的 AI Skill".

**Layout structure:**

**Top Section (Title Area - 25%):**

- Main title in bold, large font:
  "同事是怎么被蒸馏成 AI Skill 的"
- Subtitle below:
  "信息收集 → 双线抽象 → Skill 装配"
- Small side annotation:
  "高层流程总览"

**Central Section (Hero Workflow - 55%):**
Create a clean 4-step workflow diagram with strong visual flow:

1. **输入同事信息**
   - Small icons or cards for:
     - 姓名 / 岗位 / 职级
     - MBTI / 标签 / 主观印象
     - 飞书 / 钉钉 / 邮件 / PDF / 聊天记录
   - Visual style: stacked cards or source tiles

2. **进入分析引擎**
   - A central processing box labeled:
     - "同事蒸馏流程"
   - Small annotation:
     - "把零散材料变成结构化规则"
   - Show arrows splitting into two branches

3. **双线抽象**
   - Upper branch:
     - `work.md`
     - Short annotation: "工作方法 / 技术规范 / 流程经验"
   - Lower branch:
     - `persona.md`
     - Short annotation: "表达风格 / 决策模式 / 人际行为"
   - Use two clearly separated but visually symmetrical boxes

4. **装配成最终 Skill**
   - A merged output card on the right:
     - `SKILL.md`
     - `colleagues/{slug}/`
   - Small annotation:
     - "可直接调用的电子同事"
   - Add subtle indication of versioned asset / packaged artifact feeling

**Bottom Section (Summary Strip - 20%):**

Show a short horizontal summary with 3 compact phrases:

- "先收集"
- "再抽象"
- "后运行"

Or alternatively:

- "输入原材料"
- "生成 work + persona"
- "得到可调用 Skill"

Use tiny arrows between them.

**Key visual elements:**

- Source material cards on the left
- One central distillation/processing box
- Two parallel output branches in the middle
- One merged final artifact on the right
- Strong directional arrows showing the full pipeline
- Minimal but clear labels in Chinese

**Visual style:**

- Hand-drawn technical sketch aesthetic
- High-level system workflow, not dense engineering detail
- Strong visual hierarchy
- Minimal text, mostly diagrams and labeled boxes
- Clean and breathable, suitable for use as a cover image

**Color palette:**

- Navy blue (#2C3E50) for main structure and text
- Teal (#16A085) for arrows and flow highlights
- Light gray for source cards and secondary boxes
- Minimal orange (#E67E22) for emphasis on final Skill output
- Pure white background

**Typography:**

- All annotations in simplified Chinese
- Title bold and prominent
- Subtitle medium size
- Module labels clear and readable
- Keep text sparse; avoid dense paragraphs

**Overall feel:**

- Clean
- Modern
- Professional
- Easy to understand at a glance
- More like a high-level Agent workflow cover than a detailed infographic

Style: Modern technical hand-drawn cover, high-level workflow diagram, clean white background, social-media-friendly
